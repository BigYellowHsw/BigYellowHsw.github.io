PyQt5介绍及环境安装

发表于 2018-12-24 | 分类于 PyQt5 | 本文总阅读量 次

安装PyQt5及配置
第一步：安装python，pycharm,更新pip
1.安装python，记得添加路径到path中

2.更新pip

python -m pip install --upgrade pip
easy_install -U pip

3.安装pycharm

第二步 安装PyQt5以及pycharm的配置
1.安装PyQt5库
pip install PyQt5 -i https://pypi.douban.com/simple
或者上pypi上下载PyQt5的whl包安装

2.安装常用的pyqt工具库 PyQt5-tools
pip install PyQt5-tools -i https://pypi.douban.com/simple

3.pycharm配置Qt Designer和pyuic

在设置里面点击external tools，点击“+”，需要添加Qt Designer 和pyuic 两个选项。

在新增Qt Designer窗口中的填写内容：

Name：Qt Designer
program：Qt Designer的安装路径（也就是pyqt5-tools包里的 desginer.exe）
parameter：不填
directory： $FileDir$

在新增pyuic窗口中的填写内容：

Name：pyuic
program：pyuic的安装路径(也就是PyQt5\uic\pyuic.py文件)
parameter：$FileName$ -o $FileNameWithoutExtension$.py
directory： $FileDir $

第三步 安装pyinstaller

pyinstaller库的作用是将pyqt5写好的py文件打包成exe。

pip install pyinstaller

所以，利用pyqt5开发界面的正确姿势为：

1.先利用qt designer设计出界面样式ui

2.利用pyuic转换ui为py类

3.在自己的程序中导入界面py类，编辑需要的控件逻辑

4.完成后利用pyinstaller打包
