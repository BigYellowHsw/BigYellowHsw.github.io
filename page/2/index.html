python配置virtualenv环境

发表于 2018-12-23 | 分类于 python

在python实际项目开发中，我们通常会根据自己的需求去下载各种相应的框架库，如Scrapy、Beautiful Soup等，但是可能每个项目使用的框架库并不一样，或使用框架的版本不一样，这样需要我们根据需求不断的更新或卸载相应的库。直接对我们的Python环境操作会让我们的开发环境和项目造成很多不必要的麻烦，管理也相当混乱。所以我们在项目开发中，应该对不同的项目建立不同的虚拟环境，下载安装该项目需要的python版本、各种依赖库，然后再进行开发。本文就简单讲解下如何配置python的虚拟环境。

python配置virtualenv环境有两种方式：一种是cmd命令创建虚拟环境再激活，另一种方法是利用pycharm设置里添加个虚拟环境。

新的开始

发表于 2018-12-20 | 分类于 随笔

​

​ 快2019年了，回望自己的2018，可以用一个词，“惨淡”，来形容。这一年没有什么大的突破，技术仍然很平庸，始终有一种碌碌无为的感觉。所以，期望自己从2019年开始，改变自己吧。先从写博客开始吧，将自己2018年各个地方做的一些笔记，笔记本上的、其他论坛上的、看的一些书籍，都整理下来，迁移到自己的博客上来，其一是锻炼自己的写作能力，其二是希望自己不要像从前那样，学的东西过一阵子又很快地还给老师。

​ 曾经高中班主任喜欢一句话，叫“态度决定一切”，我很喜欢；本科的时候辅导员喜欢一句话，叫“细节决定成败”，我也很喜欢；以后，就拿这两句话作为激励自己的座右铭吧，“态度决定一切，细节决定成败！” 加油！
